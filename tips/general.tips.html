1. Keep Settings Tooltips ON while you get familiar with plugin options.

2. When you insert Option Values type them carefully. Wrong values make Tag Cloud Instance disappear until you correct them. If you get lost in finding a wrong value just delete that widget instance and add new one. It comes with a set of default option values.

3. If after installation of a new plugin's version your existing clouds disappear, just open already created widget instances and save them again without any changes.

4. Get clue for plugin abilities from these cloud examples.
